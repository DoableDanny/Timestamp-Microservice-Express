API Project: Timestamp Microservice

Simply add /api/timestamp/DD-MM-YYYY to the end of this URL

Example Usage:

  • https://timestamp-microservice-danny.herokuapp.com/api/timestamp/2015-12-25
  • OR:
  • https://timestamp-microservice-danny.herokuapp.com/api/timestamp/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
